This page demonstrates the basic DinoOverlay integration. The script will automatically detect images with the .editable-room class and add interactive overlays.

Script Integration:

<script> (function(){ var s=document.createElement('script'); s.src="https://cdn.dinooverlay.com/v1/dino-overlay-1.0.0.js"; s.integrity="sha384-[hash-will-be-generated]"; s.crossOrigin="anonymous"; s.async=true; s.onload=function(){ DinoOverlayLoader.init({ apiKey: 'demo-api-key', apiEndpoint: 'https://api.dinooverlay.com', debug: true }); }; document.head.appendChild(s); })(); </script>

Note: This is a demo using placeholder images. In production, replace 'demo-api-key' with your actual API key.
